# Yani_Montagenik - Визитка

Это страница-визитка для Yani_Montagenik, адаптированная для GitHub Pages.
Содержит все ссылки и информацию из оригинальной страницы Taplink.

## Структура проекта

- `index.html` - основная страница с социальными ссылками
- `style.css` - стили для красивого отображения страницы
- `CNAME` - файл для настройки кастомного домена GitHub Pages
- `legacy/` - директория с оригинальным исходным кодом (в гитигноре)
- `README.md` - этот файл документации

## Особенности дизайна

- Адаптивный дизайн, работает на мобильных устройствах
- Современные стили с градиентами и анимациями
- Все ссылки из оригинальной страницы сохранены
- Используется локальный аватар из legacy-версии

## Как использовать

1. Загрузите файлы на ваш репозиторий GitHub
2. Включите GitHub Pages в настройках репозитория
3. Настройте домен при необходимости (файл CNAME уже создан)

## Цветовая схема

- Основные цвета: градиент #667eea -> #764ba2 (сине-фиолетовый)
- Telegram link: #c2255c (бордовый)
- VK link: #ff4d89 (розовый)
- Instagram link: #a61e4d (темно-бордовый)
- FBSL link: #fcc419 (желтый)
- FBSL Telegram: #ffd43b (золотистый)
- Snowboard Fed: #748ffc (голубой)
